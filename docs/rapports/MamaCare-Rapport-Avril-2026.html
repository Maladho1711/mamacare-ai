Résumé exécutif

MamaCare AI est une PWA de santé maternelle pour la Guinée qui détecte les signes de danger chez les femmes enceintes via un questionnaire quotidien, alerte automatiquement le médecin par WhatsApp/SMS en cas d'urgence, et génère des résumés médicaux 30 jours avec Claude IA.

Points clés

  • Frontend déployé sur Vercel (Next.js 14, PWA installable, dark/light mode)
  • Backend migré de NestJS/Render vers Supabase Edge Functions (Deno) — cold start 1000× plus rapide
  • Base de données Postgres Supabase avec RLS (Row Level Security) sur toutes les tables
  • 13 règles OMS grossesse + 6 postnatal pour détection automatique des signes de danger
  • Alertes temps réel : WhatsApp → SMS fallback → Push notifications PWA
  • IA Claude Haiku : résumés 30j personnalisés + explications patientes en français

Sécurité et conformité

Mesures en place

🔒 Row Level Security

Activée sur les 6 tables. Chaque médecin ne voit QUE ses patientes. Chaque patiente ne voit QUE ses propres données.

🔑 JWT HttpOnly cookies

Protection XSS — le token n'est jamais accessible en JavaScript. SameSite=strict contre CSRF.

🛡️ Dev tokens HMAC

Les tokens de test sont signés avec DEV_MODE_SECRET (généré prod-grade). Impossible à forger.

🔐 Secrets Supabase Vault

ANTHROPIC_API_KEY, VAPID, cron secret — stockés chiffrés côté Supabase, jamais dans le code.

📋 Logs notifications

Chaque envoi WhatsApp/SMS tracé : recipient, statut, erreur — audit médical possible.

⚡ Cron protégé

Endpoint /send-daily-reminders protégé par x-internal-secret (INTERNAL_CRON_SECRET).

Conformité WCAG (accessibilité)

  • WCAG 1.4.4 — Zoom jusqu'à 5× autorisé (malvoyants)
  • Contraste AA — Texte gray-800 sur fond off-white validé
  • Focus visible — Ring rose sur tous les éléments interactifs
  • prefers-reduced-motion — Animations désactivées si préférence système
